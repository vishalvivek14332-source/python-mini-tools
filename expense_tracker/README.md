# Expense Tracker

A simple command-line Expense Tracker built with Python to record and analyze daily expenses.

## Features
- Add expenses with description, amount, and category
- View all recorded expenses
- Display the total amount spent
- Persistent storage using a JSON file

## Project Structure

```text
expense_tracker/
├── database.py
├── main.py
└── README.md
```

## Usage

```bash
cd expense_tracker
python main.py
```

Follow the on-screen menu to add, list, and summarize your expenses.

## Requirements

- Python 3.8 or later
