# 🔐 Password Generator

A simple Python GUI application that generates strong and secure passwords with customizable options.

## 🚀 Features

- Easy-to-use graphical interface
- Generate passwords of any length
- Option to include numbers
- Option to include special characters
- Random and secure password generation
- Instant password display

## 🛠 Requirements

- Python 3.x
- Tkinter (included with Python)

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/vishalvivek14332-source/python-mini-tools.git
```

Navigate to the project folder:

```bash
cd python-mini-tools/password_generator
```

Run the application:

```bash
python main.py
```

## 📋 Usage

1. Enter the desired password length.
2. Select whether to include:
   - Digits (0-9)
   - Symbols (!@#$%^&*)
3. Click **Generate Password**.
4. Copy and use the generated password.
