# 🔳 QR Code Generator

A simple Python GUI application that generates QR codes from text, URLs, or other data and saves them as PNG images.

## 🚀 Features

* User-friendly graphical interface
* Generate QR codes instantly
* Supports text, URLs, and other data
* Save QR codes as PNG images
* Lightweight and easy to use

## 🛠 Requirements

* Python 3.x
* Tkinter (included with Python)
* qrcode
* Pillow

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/vishalvivek14332-source/python-mini-tools.git
```

Navigate to the project folder:

```bash
cd python-mini-tools/qr_generator
```

Install dependencies:

```bash
pip install qrcode[pil]
```

## ▶️ Run the Application

```bash
python main.py
```

## 📋 Usage

1. Enter text or a URL.
2. Click **Generate QR Code**.
3. Choose a location to save the QR code.
4. Open the generated PNG file and scan it.
